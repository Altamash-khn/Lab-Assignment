# CustomerLabs React Assignment

This repository contains a **React** implementation of the CustomerLabs coding test (screen-1.0).  
The app builds a small **Segment Creator** UI where users can add a segment name, dynamically add schema dropdowns, and submit the result to a webhook in the required JSON format.

## Live Demo

<a href="https://lab-assignment.vercel.app/" />

## Tech Stack

- **React**: functional components + Hooks
- **Vite** dev server + build
- **Tailwind CSS**: Styling
- **Shadcn UI**: PopUp

## Installation & Local Setup

```bash
# Clone the repo
git clone https://github.com/Altamash-khn/lab-assignment.git
cd lab-assignment

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open http://localhost:5173 in your browser.
